# 更新日志

本项目遵循[语义化版本](https://semver.org/lang/zh-CN/)。

## [1.7.0] - 2026-07-06

### 新增
- **多选/批量删除**：每行文件前增加复选框，支持全选/取消全选，一键批量删除选中文件
- **聊天气泡优化**：头像圆圈（昵称首字 + 哈希着色）、操作按钮仅 hover 时显示减少视觉干扰、新消息入场动画

## [1.6.0] - 2026-06-29

### 新增
- **文件搜索/过滤**：文件列表上方搜索框，实时过滤（不区分大小写），跨分类搜索，空搜索恢复分组
- **上传通知**：右下角悬浮通知"✅ N 个文件已上传"，点击定位并高亮首个文件，3 秒后自动淡出

## [1.5.0] - 2026-06-29

### 新增
- 文件列表按类型自动分组(图片/文档/视频/音频/压缩包/其他),每类可折叠展开,清爽不杂乱
- 分类折叠状态自动记住(localStorage 持久化)

## [1.4.1] - 2026-06-29

### 修复
- 页面标题下方显示当前保存目录完整路径,避免用户找不到文件存到了哪里
- 上传后验证文件确已写入磁盘,防御静默写入失败(如磁盘满、权限不足)
- 并发上传同名文件时加锁防竞态,避免极端时序下互相覆盖
- 前端上传失败时(HTTP 非 200)现在会显示红色错误提示,不再静默忽略

## [1.4.0] - 2026-06-29

### 新增
- 开机自启动：可在设置面板中一键开启/关闭(Windows 通过 Startup 文件夹实现,无需管理员权限)
- 自定义保存目录：支持 `--dir` CLI 参数和 Web UI 配置,优先级 CLI > 环境变量 > 配置文件
- 全局配置文件 `%APPDATA%\lan-drop\config.json`,持久化自启、目录、端口、地址等设置
- Web UI 设置面板(页面底部 ⚙️ 可折叠),包含自启开关和保存目录配置

## [1.3.0] - 2026-06-29

### 新增
- Ctrl+V 粘贴上传：在页面任意位置粘贴剪贴板中的图片或文件即可上传,无需拖拽或点击选择
- 聊天消息持久化：消息自动保存到 `shared/.chat.json`,服务重启后聊天记录不丢失
- 操作日志：上传、下载、删除等操作记录到 `shared/.lan-drop.log`,方便追溯

### 增强
- 新增速率限制：按 IP 滑动窗口限流(消息 10条/10秒,上传 30次/分钟,删除 20次/分钟),防止滥用
- 文件列表自动隐藏以 `.` 开头的系统/工具文件(.chat.json、.lan-drop.log 等)

## [1.2.2] - 2026-06-08

### 新增
- 聊天室显示实时在线人数(基于 SSE 连接数),断线时标记为「已断开」

### 修复
- SSE 连接数无上限的问题,新增 50 连接上限,防止恶意或反复刷新累积常驻线程
- 极端时序下消息可能重复渲染的问题,追加消息前先去重

## [1.2.0] - 2026-06-05

### 新增
- 聊天室:文本互传升级为带昵称的实时聊天,支持气泡式 UI(自己的消息靠右蓝色、他人靠左灰色)
- 随机生成昵称(存储在浏览器 localStorage),可随时修改
- SSE (Server-Sent Events) 实时推送:新消息和删除操作即时同步到所有已连接设备,无需等待轮询
- SSE 断线自动降级为 4 秒轮询,恢复后自动重连

## [1.1.1] - 2026-06-05

### 修复
- 中文(及其他非 ASCII)文件名上传后被清空、只剩扩展名的问题。改用保留 Unicode 的文件名清洗逻辑,同时仍防御路径穿越、非法字符和 Windows 保留设备名

## [1.1.0] - 2026-06-05

### 新增
- 文本互传：在页面底部新增共享文本框，可在设备间快速传递文字、链接、验证码等。任一设备发送的文本，其他设备自动同步可见，支持一键复制
- 文本消息支持删除，`Ctrl+Enter` 快捷发送
- 文本消息每 4 秒自动刷新，与文件列表同步

### 说明
- 文本消息为内存存储，服务重启后清空（最多保留最近 200 条）

## [1.0.0] - 2026-06-05

首个发布版本。

### 新增
- 基于 Web 的局域网文件传输，浏览器即可访问，无需客户端
- 拖拽 / 点击上传，支持多文件并显示实时进度条
- 文件列表展示，每 4 秒自动刷新
- 文件下载与删除
- 同名文件自动追加序号，避免覆盖
- 启动时在终端显示局域网访问地址和二维码
- 路径穿越防护，限制访问范围在共享目录内
- 可通过 `--port` / `--host` 参数及 `LAN_SHARE_DIR` 环境变量配置
