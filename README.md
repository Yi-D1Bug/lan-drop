# 局域网文件传输

一个零配置的局域网文件传输工具。启动后，同一局域网内的任意设备（电脑、手机、平板）用浏览器打开即可上传、下载、删除文件，无需安装客户端。

## 特性

- **零客户端**：任何带浏览器的设备都能用，扫码即连
- **拖拽上传**：支持多文件，带实时进度条
- **共享投放区**：任一设备上传的文件，其他设备都能看到并下载
- **文本互传**：内置共享文本框，设备间快速传递文字、链接、验证码，一键复制
- **匿名聊天室**：随机昵称、气泡式 UI、SSE 实时推送,局域网内即时通讯
- **自动刷新**：文件列表每 4 秒自动同步,聊天消息实时推送
- **单文件部署**：核心逻辑全在 `app.py`，仅依赖 Flask
- **路径安全**：内置路径穿越防护，无法读取共享目录之外的文件

## 安装

需要 Python 3.7+。

```bash
pip install -r requirements.txt
```

## 使用

```bash
python app.py                # 默认端口 8000
python app.py --port 8200    # 自定义端口
python app.py --host 127.0.0.1 --port 8200   # 仅本机访问
```

启动后终端会打印局域网访问地址和二维码：

```
================================================
  局域网文件传输 v1.0.0 已启动
  共享目录: .../shared
  本机访问: http://127.0.0.1:8000
  局域网访问: http://192.168.1.5:8000
================================================
```

手机扫码或在浏览器输入「局域网访问」地址即可。上传的文件保存在 `shared/` 目录。

### 自定义共享目录

通过环境变量 `LAN_SHARE_DIR` 指定保存位置：

```bash
LAN_SHARE_DIR=/path/to/files python app.py
```

## ⚠️ 安全须知

本工具**不带任何鉴权**：凡是能访问到该地址的设备，都可以上传、下载和删除文件。这对于可信局域网内的临时投放区是符合预期的设计。

- ✅ 适用：家庭、办公室等可信局域网
- ❌ **切勿**将本工具暴露到公网，或在路由器上做端口映射 —— 这等于向所有人开放一个可任意读写文件的入口

如需在不完全可信的网络中使用，请自行在前置反向代理（Nginx 等）上配置访问控制。
